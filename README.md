# Humanity Constellation

Interactive UX prototypes for FIELD — modular application workspace, entity constellation, and experiment / transmute flows.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3457](http://localhost:3457) and use the entry page to jump between prototypes.

## Prototypes

| File | Description |
|------|-------------|
| `index.html` | Entry hub |
| `constellation+experiment.dc.html` | Humanity constellation + experiment entry |
| `Application Workspace.dc.html` | Common App workspace with orchestration, experiment sidebar, transmute pairs |
| `Canvas.dc.html` | Canvas shell |
| `support.js` | DC runtime |
| `experiment-runtime.js` | Shared experiment helpers |

Built with the DC (`.dc.html`) prototype runtime — no build step required beyond static serving.
